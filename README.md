# 🛍 E-commerce App

An **E-commerce web application** built with modern technologies.
Browse products, add to cart, and complete purchases — designed with a responsive UI and structured backend.

🔗 **Figma Prototype:** [https://www.figma.com/proto/B5VhJDkdq79Fghai45ywCx/Untitled?node-id=0-1&t=dGqDEm9syNweuPy8-1](https://www.figma.com/proto/B5VhJDkdq79Fghai45ywCx/Untitled?node-id=0-1&t=dGqDEm9syNweuPy8-1) ([GitHub][1])

---

## 📌 Features

✔ User authentication (Sign Up / Login)
✔ Product browsing & category filter
✔ Add to cart & cart management
✔ Product details & reviews
✔ Responsive UI
✔ Secure checkout process
✔ Admin panel to manage products

## 🛠 Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/abinaya-345/E-commerce-app.git
```

### 2️⃣ Backend Setup

```bash
cd E-commerce-app/backend
npm install
```

Create `.env`:

```
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Start Backend:

```bash
npm start
```

### 3️⃣ Frontend Setup

```bash
cd ../frontend
npm install
npm start
```

## 💻 How It Works

1. User visits the homepage and browses products.
2. Add desired products to cart.
3. Proceed to checkout (login/signup if required).
4. View and manage orders in user profile.
